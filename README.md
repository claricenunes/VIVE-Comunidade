# PatasDF

O **PatasDF** é um projeto criado para incentivar a **adoção responsável de cães e gatos no Distrito Federal**, além de conectar animais resgatados a **lares temporários voluntários & adoções**.

A plataforma busca facilitar o processo de adoção e ampliar a rede de pessoas dispostas a ajudar animais em situação de abandono, utilizando ferramentas simples, acessíveis e gratuitas.🐾

---

## 🌎 Objetivo

O projeto tem como objetivo:

- Incentivar a **adoção responsável de cães e gatos**
- Conectar **animais resgatados a lares temporários**
- Facilitar o cadastro de pessoas interessadas em **adotar ou oferecer lar voluntário**
- Criar uma solução **simples e gratuita** que possa ajudar ONGs, protetores independentes e adotantes

---

## 🐾 Como funciona

O funcionamento da plataforma é simples:

1. O usuário acessa o site do **PatasDF**
2. Escolhe uma das opções:
   - Adotar um animal
   - Oferecer lar temporário
3. Preenche um formulário com suas informações
4. As respostas são registradas automaticamente
5. Os responsáveis analisam os dados e conectam animais aos interessados

---

## 🛠️ Tecnologias utilizadas

O projeto foi desenvolvido utilizando:

- **HTML** — estrutura das páginas
- **CSS** — estilização e layout
- **JavaScript** — interatividade da interface
- **Google Forms** — coleta das informações dos usuários
- **Google Sheets** — armazenamento e organização das respostas

Essa solução permite que o projeto funcione **sem necessidade de servidor próprio ou banco de dados**, mantendo o custo **100% gratuito**.

## 🔗 Configuração dos formulários

O projeto utiliza **Google Forms** para coletar informações de adoção e de voluntários.

Para configurar:

1. Crie um formulário no **Google Forms**
2. Crie perguntas como:
   - Nome
   - Contato
   - Interesse (adoção ou lar temporário)
   - Experiência com animais
3. No Google Forms, clique em **Enviar**
4. Copie o **link de incorporação ou do formulário**
5. Cole o link no arquivo `index.html` do projeto

As respostas serão automaticamente armazenadas em uma **planilha do Google Sheets**.
